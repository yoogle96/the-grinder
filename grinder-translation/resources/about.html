The Grinder is a JavaTM load testing framework that makes it easy to run a distributed test using many load injector machines. It is freely available under a BSD-style open-source license.

The Grinder's home page is http://grinder.sourceforge.net/. There you can find how to get support, the user guide, links to download the latest version, the source code, license information, and details of related projects.
